# Study Buddy 📚

Інтерактивний тренажер для підготовки дитини до вступних іспитів у ліцей.

Симуляція реального іспиту з перевіркою відповідей через Gemini AI — не навчання з нуля, а перевірка готовності.

---

## Можливості

- Іспити з математики та української мови
- Gemini AI перевіряє кожну відповідь і дає пояснення
- Оскарження оцінки з аргументом
- Розгорнуте пояснення + міні-чат по кожному питанню
- Банк питань: генерується наперед, перемішується, не повторюється
- Слабкі теми: автоматично визначає де найбільше помилок (поріг 75%)
- Статистика: графік результатів, прогрес по темах
- Відновлення сесії після закриття
- Темна/світла тема, розмір шрифта
- Підтримка Gemini Nano (локальна валідація, офлайн)

## Способи використання

### Веб / PWA
Відкрий у браузері або встанови як додаток на комп'ютер, телефон чи планшет:

👉 **[study-buddy URL]**

### Chrome Extension (Side Panel)
1. Завантаж або склонуй репозиторій
2. Відкрий `chrome://extensions/`
3. Увімкни **Режим розробника**
4. Натисни **Завантажити нерозпаковане розширення** → обери папку проекту
5. Клікни на іконку розширення в тулбарі

## Налаштування

Для роботи потрібен **Gemini API ключ** (безкоштовно):

1. Перейди на [Google AI Studio](https://aistudio.google.com/apikey)
2. Натисни **Create API key** (при потребі створи проект)
3. Встав ключ у налаштуваннях додатку (кнопка ⚙️)

> **Порада для батьків.** Якщо є платна підписка Google One або Google Workspace — ліміти API там значно вищі. Краще створити ключ у тому акаунті.

## Стек

- Vanilla JS — без фреймворків, без npm, без build-степу
- Gemini API — генерація питань і перевірка відповідей
- Chrome Extension Manifest V3
- PWA — встановлюється як нативний додаток
- localStorage — весь прогрес зберігається локально
