Як отримати API ключ

API ключ дозволяє додатку звертатись до Gemini — мозку Study Buddy.

  1. Перейди на Google AI Studio
  2. Натисни «Create API key»
  3. Якщо проекту ще немає — Google запропонує його створити. Погодься і дочекайся завершення
  4. Скопіюй ключ і встав його у поле API ключа
Порада для батьків. Якщо у вашій родині є будь-яка платна підписка Google One або Google Workspace — ліміти Gemini API там значно вищі. У такому випадку краще створити ключ у тому акаунті, де є підписка.
